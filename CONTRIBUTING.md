# Contributing to PyPLUTO

Welcome! We're happy to have you here. Thank you in advance for your contribution to PyPLUTO.
If you have any questions, suggestions or find a bug, feel free to open an issue or fork the repository and create a pull request.
Any contribution aimed at helping the PLUTO code community to have better plots with less efforts will be greatly appreciated.

## Use of AI

The use of AI (i.e., LLMs) is not discouraged if used responsibly.
However, every developer and contributor remains responsible for the code they publish, and as PyPLUTO developers, we request that every line of code produced is checked by humans.

This rule holds also for communications with developers via e-mail, issues or pull requests. We do not discourage AI for translation purposes, but please take your time to ensure that the text matches your intentions.

Ino order to keep PyPLUTO maintainable, we do not allow autonomous agents to contribute to the code without human supervision.
